## What I Learned

In Modules 9, 10, and 11, I learned how to use functions in Python to organize a program into smaller sections. Instead of putting every calculation in the main part of the program, I practiced writing functions that receive values, perform a calculation, and return the answer back to the main program.

I learned how values can be passed into a function through parameters. For example, I passed quantity and price into a function to calculate an extended price, passed hits and at bats into a function to calculate a batting average, and passed sales information into a function to calculate a forecast. This helped me understand how functions make programs easier to read and reuse.

I also learned how to return values from a function. Some programs returned one value, such as a total price or batting average. Other programs returned two values, such as a discount amount and discounted price, or total exam points and average score. This helped me practice using multiple return values in Python.

Another skill I practiced was using loops with functions. The programs repeatedly ask the user whether they want to continue. If the user enters `Yes`, the program collects more data, calls the function, and displays the result. If the user enters anything else, the loop stops. This showed me how loops and functions can work together to process more than one record.

I also continued practicing decision-making with `if`, `elif`, and `else` statements inside functions. Some functions used conditions to choose a discount rate, forecast percent, or price calculation before returning the result. This made the programs more realistic because the output depends on the input.
